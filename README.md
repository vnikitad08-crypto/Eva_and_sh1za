# EvoChat

Клиентский мод для **Minecraft Fabric 1.21.4** под сервер **DiamondWorld (PrisonEvo)**.
Объединяет удобное окно личных сообщений, замену никнеймов, подсветку игроков по кланам,
кастомные скины, руны, автосообщение и таймер ивентов — всё в одном меню в фирменном
зелёном стиле.

**Авторы:** Eva_Elfie и sh1zaExE

> ⚠️ Мод **только клиентский** и косметический. Он ничего не отправляет и не меняет на
> сервере (кроме команд, которые ты запускаешь сам). Всё, что связано с внешним видом
> — скины, подсветка, замена ников — **видно только тебе**; другие игроки этого не видят.

---

## Требования

| Компонент | Версия |
|---|---|
| Minecraft | **1.21.4** |
| Fabric Loader | **0.17.2** и новее |
| Fabric API | любая под 1.21.4 |
| Java | **21+** |
| Сторона | только клиент |

Для кастомных **скинов** дополнительно нужен ресурс-пак сервера **DiamondWorld** (модели
берутся из него). Без пака подмена скинов просто не будет видна.

## Установка

1. Установи **Fabric Loader 0.17.2+** для Minecraft 1.21.4.
2. Положи в папку `mods`:
   - `evochat-1.2.0.jar`
   - **Fabric API**
3. Запусти игру. Меню мода открывается клавишей **`]`** (Ъ на русской раскладке).

Файл `evochat-1.2.0-sources.jar` в `mods` класть **не нужно** — это исходники.

## Возможности

### 💬 Виджет ЛС (личные сообщения)
- Отдельное HUD-окно для личных сообщений: сообщения с меткой «ЛС» дублируются в окно,
  оставаясь и в обычном чате.
- Прокрутка колесом мыши над окном или клавишами `PageUp` / `PageDown`, полоса прокрутки,
  перенос длинных строк, счётчик сообщений, очистка истории.
- Настройка прозрачности текста и фона, ширины и высоты окна.
- **Прозрачность всего окна Minecraft** (через GLFW) — сквозь игру виден рабочий стол.
- Скрывается по `F1` вместе с остальным HUD.

### 🔔 Уведомления
- Отдельное меню: **звук** нового ЛС (свой независимый канал, не зависит от громкости
  Minecraft) и **ползунок громкости**.

### 🏷️ Замена никнеймов
- Замена ника любого игрока на любой другой, который увидишь **только ты**.
- Работает в **чате, над головой (nametag) и в списке игроков (Tab)**. В Tab замена
  идёт по точному нику аккаунта — надёжно.
- Поддержка **цветовых кодов** `&` (`&4`, `&a`, `&l`, `&r` и т.д.) — ник можно раскрасить.
- Меню с добавлением, редактированием, удалением замен, общий переключатель вкл/выкл,
  цветной предпросмотр.

### 🛡️ Кланы и подсветка игроков
- **Список кланов**: добавление клана вручную, счётчик игроков, просмотр состава,
  удаление.
- Для каждого клана — отношение: **Враг** / **Друг** / **Нейтрал**.
- Подсветка игрока в мире по отношению его клана:
  - **Враг** — красный (можно задать свой цвет),
  - **Друг** — зелёный (можно задать свой цвет),
  - **Нейтрал** — без подсветки.
- Подсвечивается **вся модель игрока — тело и броня**, с обычным тестом глубины
  (**не сквозь стены**).
- **Ползунок прозрачности подсветки**: слева (0%) — игрок полностью в цвете (яркая
  сплошная заливка, светится даже в тени), справа (100%) — подсветки нет.
- Общий переключатель подсветки в главном меню.

### 🟢 Проверка онлайна
- Проверка игроков через личные сообщения (`/tell`): распознаёт «оффлайн», «онлайн»,
  «отключил ЛС», а также «этот игрок вас **игнорирует**» (= онлайн).
- Проверка идёт с паузой между отправками, чтобы не спамить сервер пакетами.
- Кнопки «Проверить все» / «Стоп».

### ✉️ Автосообщение
- Автоматическая отправка сообщения в чат по интервалу (минуты/секунды), с текстом,
  экранным виджетом статуса и переключателем по клавише `F8`.

### 🗡️ Кастомные скины (только у тебя)
- **Меч** — 20 скинов.
- **Призматический меч** — 12 тем × стадии улучшения.
- **Броня** — 26 комплектов (применяется ко всей надетой броне).
- Скин применяется **только к твоим предметам**, на других игроках не показывается.
  Требуется ресурс-пак сервера.

### 🔮 Руны
- Быстрое применение сетов рун и открытие «мешка рун» (через команды `runesbag` / `runeset`).

### ⏱️ Таймер ивентов DiamondWorld
- HUD-обратный отсчёт до серверных ивентов (Мифический ивент, Золотая лихорадка).
  Можно включить/выключить в главном меню.

### 🧩 Редактор HUD-виджетов
- Перетаскивание мышью — свободное расположение окна ЛС, виджета автосообщения и таймера
  ивентов на экране.

### 📊 Сбор данных и экспорт
- Мод собирает игроков из чата (ник / клан / уровень), группирует по кланам.
- Экспорт в текстовый файл, очистка, ручное добавление/удаление игроков.

## Главное меню (клавиша `]`)

Панель в зелёном стиле с заголовком **EvoChat**:

- **Никнеймы** — замена ников
- **Список кланов** — кланы, отношения, подсветка, онлайн-проверка
- **Скины** — Меч / Призма / Броня
- **Клавиша: …** — переназначить клавишу открытия меню
- **Виджеты HUD** — перетаскивание виджетов
- **Настройки** (широкая) — единый экран настроек с вкладками:
  - *Основные* — команда для ЛС, сообщение для проверки, символ клан-чата, таймер ивентов
  - *Чат* — прозрачность/размер окна ЛС, прозрачность окна Minecraft, кнопка «Уведомления»
  - *Автосообщение*
- Переключатели: **Подсветка кланов (не сквозь стены)**, **Таймер ивентов DW**
- **Экспорт данных** / **Очистить данные**

## Горячие клавиши

| Клавиша | Действие |
|---|---|
| `]` (Ъ) | Открыть меню мода |
| `F8` | Автосообщение: вкл/выкл |
| `PageUp` / `PageDown` | Прокрутка окна ЛС |

Все клавиши мода можно переназначить в **Настройки → Управление** Minecraft (категория
**EvoChat**).

## Команды

Корень — `/chatlogger`:

| Команда | Действие |
|---|---|
| `export [клан]` | экспорт данных (всех или клана) |
| `list [клан]` | список игроков в чат |
| `clans` | список кланов |
| `count` | количество игроков |
| `check [клан]` / `stop` | проверка онлайна / стоп |
| `add <ник> [ур.] <клан>` / `remove <ник>` | добавить / убрать игрока |
| `clear` / `clean` | очистить всё / убрать неверные ники |
| `highlight enable/disable/color/remove/list/enemy/friend` | управление подсветкой |
| `sendclan <клан>` | отправить состав клана в клан-чат |
| `runesbag` / `runeset <1-5>` | мешок рун / применить сет рун |
| `gui` | открыть меню |
| `help` | полный список команд |

## Файлы конфигурации

Создаются автоматически в папке `.minecraft`:

- `chatlogger_data.txt` — собранные игроки и кланы
- `chatlogger_highlight_config.properties` — отношения кланов, цвета, прозрачность подсветки
- `chatlogger_nicknames.properties` — замены ников
- `chatlogger_skins.properties` — выбранные скины
- + настройки чата/автосообщения/рун

## Сборка из исходников

Требуется **JDK 21** и интернет (для загрузки Minecraft/Fabric).

```bash
./gradlew build      # Linux/macOS
gradlew.bat build    # Windows
```

Готовый мод: `build/libs/evochat-1.2.0.jar` (файл `*-sources.jar` не нужен).

> Косметические миксины (`ItemSkinMixin`, `ArmorSkinMixin`) — необязательные: если в
> другом маппинге целевой метод назван иначе, они просто не применятся, игра не упадёт,
> остальной мод продолжит работать.
